## Если что-то не работает

### Инвентарь не загружается
```
1. Проверьте интернет
2. Проверьте авторизацию в Firebase
3. Перезагрузите приложение
```

### ХП/АС не видно
```
1. Обновите приложение (flutter pub get)
2. Очистите кэш (flutter clean)
3. Пересоберите (flutter run)
```

### Баг с расой вернулся
```
Это невозможно - баг исправлен в коде
Просто закройте и откройте приложение
```

## Типичные вопросы

**Q: А мои старые предметы?**
> A: Если они были сохранены в Firestore, они загружаются. Если нет - они были потеряны (это была проблема, которую мы исправили).

**Q: Что, если отключится интернет?**
> A: Все работает локально. Синхронизация произойдет, когда интернет вернется.

**Q: Как часто синхронизируется?**
> A: Моментально при добавлении/изменении предмета. Нет задержки.

**Q: Где хранятся данные?**
> A: В двух местах: локально (Hive) и в облаке (Firestore).

**Q: Могу ли я удалить локальные данные?**
> A: Не рекомендуется. Используйте встроенные функции приложения.

## Для тестирования

### Быстрый тест (2 минуты):
```bash
1. flutter clean
2. flutter pub get
3. flutter run
4. Добавьте предмет
5. Закройте приложение (Force Stop)
6. Откройте приложение
7. Проверьте, что предмет остался ✓
```

### Полный тест (5 минут):
```bash
1. Сделайте быстрый тест
2. Отключите интернет (Airplane mode)
3. Добавьте еще предмет
4. Закройте приложение
5. Откройте приложение
6. Проверьте, что оба предмета есть ✓
7. Включите интернет
8. Проверьте синхронизацию в Firebase Console ✓
```

## Архитектура хранения данных

### HIVE (локальное):
```
Box 'characters':
  Key: character.name
  Value: Character JSON

Box 'inventories':
  Key: character.id или character.name
  Value: Inventory JSON (список предметов)

Box 'item_database':
  Key: 'items'
  Value: ItemDatabase JSON (вся база предметов)
```

### FIRESTORE (облако):
```
users/{userId}/characters/{charId}/
  ├─ id, name, level, hp, ac, stats...
  └─ inventory/data
      └─ items: [{item1}, {item2}, ...]
         updatedAt: timestamp
```

## Если что-то не работает

### Инвентарь не сохраняется:
1. Проверьте logcat на сообщения ✅ СОХРАНЕНО
2. Проверьте что character.id != null
3. Проверьте Firebase авторизацию (для облака)

### База предметов не сохраняется:
1. Проверьте logcat на сообщения ✅ СОХРАНЕНО В HIVE!
2. Проверьте что Hive инициализирован в main.dart
3. Проверьте что ItemDatabaseScreen загружает базу

### Предметы теряются после перезагрузки:
1. Проверьте logcat - должны быть логи загрузки
2. Если нет логов - база не загружается
3. Откройте Firebase Console → Firestore для проверки облака

# 📋 Инструкции по внедренным функциям

## 1️⃣ Отображение ХП и КБ (AC)

### Где это видно?
Откройте приложение → Выберите персонажа → Вкладка "Характеристики"

**На этой вкладке в верхней части вы увидите две информационные карточки:**

```
┌─────────────────────────────────────────┐
│  HP (Здоровье)        AC (Класс брони) │
│   [15]                    [14]          │
│  красная               синяя            │
└─────────────────────────────────────────┘
```

- **HP** отображается в красной карточке (слева)
- **AC** отображается в синей карточке (справа) и рассчитывается автоматически на основе надетой брони

## 2️⃣ Исправленный баг с расой

### Проблема была:
Если установить характеристику 20 и выбрать расу с бонусом (например, +2), то итог был 22 (что превышает лимит 20).

### Как работает сейчас:
1. Откройте "Создание персонажа"
2. Установите Ловкость = 20
3. Нажмите на "Выбрать расу" и выберите полурослика (+2 к ловкости)
4. **Результат:** Ловкость автоматически станет 18 (18 + 2 = 20) ✅

**Этот процесс происходит автоматически при выборе расы!**

#### Технически:
Когда вы выбираете расу, система проверяет каждую характеристику:
- Если (база + бонус расы) > 20
- То база = 20 - бонус расы

Таким образом, максимум всегда = 20.

## 3️⃣ Сохранение инвентаря

### Как это работает?

#### При входе в персонажа:
1. Приложение проверяет **локальное хранилище** (Hive) - **быстро** ⚡
2. Если там нечего нет, загружает из **облака** (Firestore) ☁️
3. После загрузки из облака - сохраняет локально для следующего входа

#### При добавлении/изменении предметов:
1. Вы добавляете/удаляете/редактируете предмет в инвентаре
2. Система **автоматически сохраняет** в оба места:
   - Локально (Hive) - моментально
   - В облако (Firestore) - почти моментально

### Результат:
✅ Предметы сохраняются при закрытии приложения
✅ Предметы появляются при открытии приложения
✅ Синхронизация между разными устройствами (если вход с одного аккаунта)
✅ Кэширование локально для быстрого доступа

### Проверка:

#### Локальное хранилище (Hive):
```
Директория: app/lib/
Хранилище: Hive boxes
Данные: Всегда доступны, даже без интернета
```

#### Облачное хранилище (Firestore):
```
Путь: users/{userId}/characters/{characterId}/inventory/data
Данные: Видны в Firebase Console
```

## 🐛 Возможные проблемы и решения

### Проблема: Инвентарь не загружается
**Решение:**
1. Проверьте интернет-соединение
2. Проверьте, что пользователь авторизован в Firebase
3. Посмотрите логи в Firebase Console

### Проблема: Инвентарь не сохраняется
**Решение:**
1. Проверьте, что Firestore доступна
2. Убедитесь, что правила безопасности Firestore позволяют запись
3. Проверьте локальное хранилище (Hive должен быть инициализирован)

### Проблема: Медленная синхронизация
**Решение:** Это нормально, если используется медленный интернет. Данные всегда сохраняются локально мгновенно, облако обновляется в фоне.

## Потенциальные проблемы и решения

### Проблема: Callback не вызывается
```
Симптомы: Нет логов о callback
Причины:
  1. Callback не передан в ItemDatabaseScreen
  2. widget.onItemChanged == null
  3. Вызов за пределами контекста
Решение: Проверить строку 556 inventory_screen.dart
```

### Проблема: Hive не сохраняет
```
Симптомы: Логи говорят ✅, но данные теряются после перезагрузки
Причины:
  1. Box не инициализирован в main.dart
  2. Неправильный ключ для Box
  3. JSON ошибка при кодировании
Решение: Проверить main.dart и StorageService.saveInventory()
```

### Проблема: Firestore не сохраняет
```
Симптомы: Лог "⚠️  Не удалось сохранить в Firestore"
Причины:
  1. userId == null (не авторизован)
  2. charId == null (character.id не установлен)
  3. Нет прав в Firestore
Решение: Проверить Firebase авторизацию и правила
```
